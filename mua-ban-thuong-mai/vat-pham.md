## 🍳 Món Ăn Nấu Sẵn

Các món nấu từ nguyên liệu nông sản. Món càng cao cấp thì hồi phục càng nhiều.

| Tên Món Ăn          | Hồi Phục         | Cách Nấu                       |
| ------------------- | ---------------- | ------------------------------ |
| Bắp Cải Xào Trứng   | Khoảng 20% độ no | Bắp cải + Trứng gà + Muối biển |
| Súp Bí Ngô          | Khoảng 30% độ no | Bí ngô + Sữa + Muối biển       |
| Gà Ủ Muối           | Khoảng 45% độ no | Thịt gà + Muối biển            |
| Bánh Pudding Bí Ngô | Khoảng 55% độ no | Bí ngô + Sữa + Trứng           |
| Thỏ Quanh Lửa Hồng  | Khoảng 65% độ no | Thịt thỏ + Muối biển           |
| Thỏ Hầm Bí Ngô      | Khoảng 70% độ no | Thịt thỏ + Bí ngô + Muối biển  |

**Dụng cụ nấu ăn:**
| Loại Bếp | Đặc Điểm | Cách Nhận |
|---|---|---|
| Bếp Củi | Thủ công, nấu được vài món | Chế tạo |
| Bếp Gas | Nấu nướng nhanh, nấu được nhiều món hơn | Chế tạo |
| Bếp Điện Từ | Hiện đại, hâm nóng nhanh, nấu được nhiều món hơn | Chế tạo |

## 👕 Quần Áo & Ngoại Hình

Tất cả các trang phục và phụ kiện giúp tùy chỉnh ngoại hình nhân vật. Bạn thay đồ tại **tủ quần áo** tại nhà hoặc tại **cửa hàng quần áo**.

| Loại Trang Phục                           | Mô Tả                              |
| ----------------------------------------- | ---------------------------------- |
| Áo Khoác                                  | Mặc ngoài, đa dạng kiểu dáng       |
| Áo Thun                                   | Thường là vị trí của các loại quần |
| Quần                                      | Thường là vị trí của các loại chân |
| Giày                                      | Dép, giày thể thao, boot           |
| Mặt Nạ                                    | Che mặt, một số nghề bẩn yêu cầu   |
| Kính                                      | Kính mát trang trí                 |
| Mũ / Nón                                  | Phụ kiện trang trí                 |
| Áo Chống Đạn                              | Giáp bảo vệ cơ thể                 |
| Dây Chuyền, Khuyên Tai, Vòng Tay, Đồng Hồ | Phụ kiện trang sức                 |
| Túi Xách                                  | Giúp tăng sức chứa                 |

**Phụ kiện quản lý:**

| Tên Vật Phẩm   | Công Dụng                                |
| -------------- | ---------------------------------------- |
| Chùm Chìa Khóa | Chứa nhiều chìa khóa xe, giữ túi gọn hơn |
| Ví Giấy Tờ     | Chứa bằng lái, căn cước và giấy tờ       |
| Vali           | Đựng quần áo, thay đồ mọi nơi            |
