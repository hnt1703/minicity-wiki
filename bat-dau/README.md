Chào mừng bạn đến với MINICITY SS2! Phần này sẽ hướng dẫn bạn những bước đầu tiên khi vào game.

- [Quần Áo & Ngoại Hình](quan-ao-ngoai-hinh.md) — Thay đổi trang phục
- [Điện Thoại](dien-thoai.md) — Sử dụng điện thoại trong game

## Những Điều Cần Biết

### Phím Tắt Cơ Bản

| Phím | Chức Năng |
|------|-----------|
| `F1` hoặc `M` | Mở menu radial |
| `Tab` hoặc `F2` | Mở túi đồ |
| `E` | Tương tác (ox_target) |
| `F3` | Điện thoại |
| `Esc` | Menu chính |

### Bắt Đầu Thế Nào?

1. **Tạo nhân vật** — Chọn ngoại hình, tên tuổi
2. **Mặc quần áo** — Tìm cửa hàng quần áo
3. **Làm việc** — Tìm nghề phù hợp để kiếm tiền
4. **Khám phá** — Di chuyển quanh bản đồ, thử các tính năng
