# Gift Code

Gift Code là mã code đặc biệt giúp bạn nhận quà rewards miễn phí. Theo dõi fanpage và Discord server để nhận code mới!

## Cách Sử Dụng

### Mở Menu
- Mở menu hệ thống
- Chọn **"Gift Code"**

### Nhập Code
1. Nhập mã code vào ô nhập
2. Nhấn **"Xác Nhận"**
3. Nhận quà ngay lập tức!

## Nơi Lấy Code

| Nguồn | Mô Tả |
|-------|-------|
| **Fanpage Server** | Theo dõi fanpage trên Facebook |
| **Discord Server** | Tham gia Discord để nhận code |
| **Sự Kiện Đặc Biệt** | Code trong các event lớn |
| **Livestream** | Code phát trong livestream |

## Lưu Ý

- **Code có thời hạn** — Sử dụng ngay khi nhận
- **Mỗi code chỉ dùng 1 lần** — Không thể dùng lại
- **Code giới hạn số lượng** — Nhanh tay nhập trước khi hết!

## Mẹo

1. **Theo dõi fanpage** — Đây là nơi phát code nhiều nhất
2. **Tham gia Discord** — Nhận code và cập nhật tin tức
3. **Nhập code ngay** — Đừng để code hết hạn
4. **Chia sẻ với bạn bè** — Giúp bạn bè cùng nhận quà
5. **Kiểm tra thường xuyên** — Code mới xuất hiện liên tục
