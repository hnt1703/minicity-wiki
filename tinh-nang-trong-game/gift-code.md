# Gift Code

Nhập mã code để nhận quà rewards.

## Cách Sử Dụng

### Mở Menu
- Mở menu
- Chọn "Gift Code"

### Nhập Code
1. Nhập mã code vào ô
2. Nhấn "Xác Nhận"
3. Nhận quà ngay!

## Nơi Lấy Code

- Fanpage server
- Discord server
- Sự kiện đặc biệt
- Livestream

## Mẹo

- Theo dõi fanpage để nhận code mới
- Code có thời hạn sử dụng
- Nhập code ngay khi có
