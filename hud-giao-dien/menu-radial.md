# Menu Radial

Menu nhanh để truy cập các tính năng.

## Mở Menu

- Nhấn **M** hoặc **F1**

## Các Chức Năng

| Chức Năng | Mô Tả |
|-----------|-------|
| Garage | Gọi xe từ garage |
| Điện Thoại | Mở điện thoại |
| Menu Chính | Truy cập menu |
| Map | Xem bản đồ lớn |
| Inventory | Mở túi đồ |
| Settings | Cài đặt |

## Sử Dụng

1. Mở menu
2. Di chuyển chuột đến chức năng muốn
3. Nhấn chọn
4. Thực hiện thao tác

## Mẹo

- Ghi nhớ phím tắt
- Sử dụng menu khi cần thao tác nhanh
- Đóng menu khi xong
