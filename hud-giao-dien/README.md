# HUD & Giao Diện

Hệ thống giao diện người dùng (HUD) hiển thị thông tin quan trọng trên màn hình. Bao gồm thanh trạng thái, bản đồ nhỏ, thông tin xe, và các menu điều khiển.

## Tổng Quan Hệ Thống

### 🖥️ HUD Chính
- **Thanh trạng thái:** Máu, đói, khát, stress
- **Thông tin nhân vật:** Avatar, nghề nghiệp, số dư
- **Thông tin xe:** Tốc độ, nhiên liệu, đèn
- **Bản đồ nhỏ:** Vị trí hiện tại

### 🎯 Menu Radial
- **Phím tắt:** `F1`
- **Tính năng:** Truy cập nhanh các menu
- **Tùy chỉnh:** Theo nghề nghiệp và cấp bậc

### 🎒 Túi Đồ
- **Phím tắt:** `Tab`
- **Quản lý:** Vật phẩm, vũ khí, trang bị
- **Giao diện:** Kéo thả

## Lệnh Hệ Thống

| Lệnh | Chức Năng |
|------|-----------|
| `/hud` | Mở cài đặt HUD |
| `/togglehud` | Ẩn/hiện HUD |

## Mẹo Sử Dụng

1. **Tùy chỉnh HUD:** Sử dụng `/hud` để chỉnh layout
2. **Ghi nhớ phím tắt:** Sử dụng phím tắt để thao tác nhanh
3. **Theo dõi trạng thái:** Luôn chú ý thanh đói, khát
4. **Ẩn HUD khi chụp ảnh:** Dùng `/togglehud` để ẩn giao diện
5. **Sử dụng menu radial:** Truy cập nhanh các tính năng
